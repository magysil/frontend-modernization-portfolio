# Frontend Modernization Portfolio
> Colección de 9 proyectos web interactivos migrados y rediseñados utilizando metodologías modernas, Bootstrap 5.3.3 y JavaScript ES6+ nativo sin dependencias de jQuery.

### 🚀 Tecnologías Globales
- HTML5 / CSS3 / JavaScript (ES6+)
- Bootstrap 5 (UI/UX Modernization)
